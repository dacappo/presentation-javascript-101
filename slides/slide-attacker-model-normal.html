Attacker Model


Attacker Model - Mightiness


The attacker is aware of the deployed
technology stack including application server, driver, frameworks and database.


The attacker is able to send arbitrary requests to the server with the authorization of a normal application user.

Attacker Model - Goal


The attacker's goal is to achieve
unintended behavior of the database query by altering query parameters.


The attacker is able to break the
CIA triad with regard to the stored data.
